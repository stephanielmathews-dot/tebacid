# tebacid
Tool for bacterial identification in Tiny Earth Lab
Biochemical characterization information for 30 common soil bacteria was used to create a database based on Bergey's Manual of Determinitive Microbiology. 
Key tests are: Gram stain,	Catalase,	Oxidase,	Gelatinase,	Glucose,	Shape,	Colony Morphology,	Hydrogen sulfide, 	Indole,	Motility,	Lactose,	Mannitol,	Hemolysis, 
Use Google Gemini Canvas this databse was used to create a JSON file and then code for this identification tool.
